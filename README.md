# KOZMİK ROTA: SOLAR MISSION

Bu klasör, KOZMİK ROTA: SOLAR MISSION oyununun yapay zekâ destekli analiz ve geliştirme çalışmalarında kullanılacak ana proje paketidir.

## Başlangıç dosyaları

- `docs/KOZMIK_ROTA_SOLAR_MISSION_PROJECT_DOCUMENT.md`  
  Ana oyun tasarım ve proje dokümanı.

- `AI_CONTEXT.md`  
  Yapay zekânın projeyi nasıl yorumlaması gerektiğini belirleyen kısa bağlam dosyası.

- `config/game_design.json`  
  Kesinleşmiş temel kararların makine tarafından kolay okunabilir özeti.

- `assets/player/Altay_X_16_bit_multi_view.png`  
  Altay-X için mevcut yan görünüş referansı.

## Kullanım

Bu klasörü kod deposunun kök dizinine ekleyin. Kod üreten yapay zekâya önce `AI_CONTEXT.md`, ardından ana proje dokümanını okutun.

## İlk hedef

Merkür bölümünü içeren tam oynanabilir MVP.
